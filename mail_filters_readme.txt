MailFilters Class adds few useful filtering features to the ImapMailbox class (Something similar to that of Gmail filters)

Features:
 * Apply filters to any of your ImapMailbox messages on the following fields:
 	- To
 	- Subject
 	- From
 * Take actions based on matched filters:
 	- Delete message from your mailbox.
 	- Move the message to a different mailbox.
 	- Return user specified variables.


In Progress (Unfinished work):

 * Filter messages that contains attachments
